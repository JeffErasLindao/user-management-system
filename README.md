# user-management-system
This project is a full-stack CRUD application focused on managing efficiently users, roles and permissions.


## Seguridad de Base de Datos
La aplicación utiliza un usuario de base de datos con permisos limitados (`app_user`)
para las operaciones CRUD. Las tareas administrativas se realizan con un usuario
administrador independiente, siguiendo buenas prácticas de seguridad.
